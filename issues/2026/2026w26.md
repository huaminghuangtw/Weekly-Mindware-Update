# Things I Learned This Week

* [駁坎](https://www.google.com/search?q=駁坎)：指在山坡地、河岸等有高低落差地形處，為防止土壤流失、邊坡崩塌所建造的護岸或擋土牆。
* i18n (Internationalization) → The framework preparation (designing for adaptation); l10n (Localization) → The act of adapting the prepared product for a specific locale (translating text, adapting images).
* A [smoke test](https://www.google.com/search?q=smoke+test) (冒煙測試) is a rapid, preliminary software testing process used to verify that the core functionality of a new build is stable, catching major bugs early. It covers essential workflows rather than in-depth scenarios, ensuring the application works at a high level.
* A **mega menu** is a type of navigation menu that displays a large number of options, often with subcategories. It’s a larger, more feature-rich version of a standard dropdown menu. Mega menus are commonly used on B2B or SaaS websites.
* 隱刺蟲 = Rove Beetles
